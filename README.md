## 🔒 Security Notice
The Firebase configuration files (`firebase_options.dart`, `google-services.json`, and `GoogleService-Info.plist`) have been excluded from this public repository using `.gitignore` to protect cloud database credentials. 

To run this project locally, please generate your own configuration files via the FlutterFire CLI or insert your respective Firebase setup files into the `lib/`, `android/app/`, and `ios/Runner/` directories.
